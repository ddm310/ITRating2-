# Архитектурные жемчужины Витебска

Лендинг, посвященный уникальной архитектуре и культурному наследию города Витебска. Проект демонстрирует исторические здания и предлагает экскурсии по знаковым местам.

## Обоснование тематики
Дизайн выполнен в спокойных, природных тонах (зеленый, бежевый), что подчеркивает гармонию городской архитектуры с природой Витебска (река Двина, парки).
* **Стилистика:** Минимализм с акцентом на типографику (шрифт Raleway).
* **Контент:** Разделение на исторические эпохи (Старый город, Задвинье, Современность) отражает богатую историю города.

## Технологии
* **HTML5:** Семантическая верстка, валидный код.
* **CSS3:** Flexbox, адаптивная верстка (Mobile First подход), CSS-анимации.
* **JavaScript (ES6):** * Табы (переключение категорий экскурсий).
  * Валидация форм.
  * Отправка заявок в Telegram Bot API.
  * Динамическая подстановка alt-тегов для доступности.
